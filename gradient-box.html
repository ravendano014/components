GradientBox

Create gradient for any elements.

About

New in 4.4.1, major updated in 4.4.2

If you need to create gradient for element background, you can use component gradient-box. To use this component, you must add role gradient-box to element. You can define linear and radial gradients. 


                    <div data-role="gradient-box">
                    ...
                    </div>

Linear gradient

To define type of gradient, you must use attribute data-gradient-type. To define linear gradient user value linear. This value is default for gradient box component and no required for defining. 


                    <div data-role="gradient-box" data-gradient-type="linear">
                    ...
                    </div>

Linear gradient position

To change gradient position, you must use attribute data-gradient-position. You can set gradient directions as angle or position. For angle, set value as angle with deg - data-gradient-position="270deg", to set gradient position, set value for data-gradient-position as one of: top (or "to top"), left (or "to left"), bottom (or "to bottom"), right (or "to right"), top left (or "to top left"), top right (or "to top right"), bottom left (or "to bottom left"), bottom right (or "to bottom right"). 


                    <div class="demo-box square" data-role="gradient-box" data-gradient-position="45deg"></div>
                    <div class="demo-box square" data-role="gradient-box" data-gradient-position="left"></div>
                    <div class="demo-box square" data-role="gradient-box" data-gradient-position="bottom right"></div>

Radial gradient

To define radial gradient, you must use attribute data-gradient-type="radial". 


                    <div data-role="gradient-box" data-gradient-type="radial">
                    ...
                    </div>

Radial gradient shape

You can set two type of radial gradient: circle and ellipsis (default). To set type of radial gradient you must use attribute data-gradient-shape="circle|ellipsis". 


                    <div data-role="gradient-box" data-gradient-type="radial" data-gradient-shape="ellipsis"></div>
                    <div data-role="gradient-box" data-gradient-type="radial" data-gradient-shape="circle"></div>

Radial gradient size

You can set radial gradient size with attribute data-gradient-size. For this attribute you can use one of: closest-side, closest-corner, farthest-corner. 


                    <div class="demo-box"
                         data-role="gradient-box"
                         data-gradient-type="radial"
                         data-gradient-size="farthest-side"
                         data-gradient-position="20px 30px"></div>

Radial gradient position

data-gradient-position = [
[ left | center | right | top | bottom | length-percentage ] |
[ left | center | right | length-percentage ] [ top | center | bottom | length-percentage ] |
[ center | [ left | right ] length-percentage? ] && [ center | [ top | bottom ] length-percentage? ]
]

Gradient colors

Color-stop points are positioned on a virtual gradient ray that extends horizontally from the center towards the right. Percentage-based color-stop positions are relative to the intersection between the ending shape and this gradient ray, which represents 100%. Each shape is a single color determined by the color on the gradient ray it intersects. To define gradient colors you must use attribute data-gradient-colors. 


                    <div class="demo-box"
                         data-role="gradient-box"
                         data-gradient-type="radial"
                         data-gradient-size="farthest-corner"
                         data-gradient-position="40px 40px"
                         data-gradient-colors="#f35 0%, #43e 100%"></div>
