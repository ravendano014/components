Components API

About

Each components have an API to interact with it.

Access to API

To get access to components API, you must call method Metro.getPlugin(...). This method receive two arguments: element (can be selector, m4q object, jQuery object or HTMLElement) and component name.

Init component

                    <div data-role="charms" id="left-charm">
                        ...
                    </div>
or

                    <div id="left-charm">
                        ...
                    </div>

                    Metro.makePlugin("#left-charm", charms, {});
Get API

                    var charm = Metro.getPlugin("#left-charm", "charms");
                    charm.open(); // API method
                    ...
                    charm.open(); // API method
