Tokenizer

Split your text into separate elements to style them individually.

About

New in 4.4.0

Metro UI 

                    <style>
                        #example1 {
                            font-size: 38px;
                            font-weight: bold;
                            color: #ffffff;
                            display: flex;
                            justify-content: center;
                            padding: 50px;
                            background: #3e372d;
                            position: relative;
                        }
                        #example1 .char {
                            font-weight: bold;
                            display: inline-block;
                            width: 48px;
                            height: 56px;
                            line-height: 56px;
                            text-align: center;
                            text-transform: uppercase;
                            z-index: 2;
                            opacity: .8;
                        }
                        #example1 .char.te-odd {
                            background: orangered;
                            transform: skewY(-20deg);
                        }
                        #example1 .char:not(.space).te-even {
                            background: orange;
                            transform: skewY(20deg);
                        }
                    </style>
                    <div id="example1" data-cls-token="char" data-role="tokenizer">
                        Metro UI
                    </div>

How it works


                    <div data-role="tokenizer">
                        Your text
                    </div>

Will transform to 


                    <div data-role="tokenizer">
                        <span>Y</span>
                        <span>o</span>
                        <span>u</span>
                        <span>r</span>
                        <span></span>
                        <span>t</span>
                        <span>e</span>
                        <span>x</span>
                        <span>t</span>
                    </div>
