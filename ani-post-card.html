Post Card

Use builtin animation for your object to create post card animation.

About

To create post-card animation you must create container with class post-card and add two inner blocks with classes .left and .right. Also you must define size for post-card: height and width 


                    <div class="post-card" style="height: 340px; width: 480px">
                        <div class="left"></div>
                        <div class="right"></div>
                    </div>


Activation effect

To activation post-card effect, you must add one of two subclasses:

  • .effect-on-hover - card rotated on user hover cursor on the card
  • .effect-on-active - card rotated on toggle class .active on card

effect-on-hover


                    <div class="post-card effect-on-hover">
                        <div class="left"></div>
                        <div class="right"></div>
                    </div>

effect-on-active


                    <div class="post-card effect-on-active">
                        <div class="left"></div>
                        <div class="right"></div>
                    </div>

Left or Right

By default, card rotated to right side. To rotate card to left side, add subclass .rotate-to-left to card. 


                    <div class="post-card effect-on-hover rotate-to-left">
                        <div class="front"></div>
                        <div class="back"></div>
                    </div>
