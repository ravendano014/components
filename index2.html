ABOUT METROUI

MetroUI is an open-source frontend toolkit inspired by Microsoft Fluent (former Metro) design principles. You can consider it as a Bootstrap replacement for web sites that need a fresh and catchy look.
MetroUI includes everything you need to build your frontend app: responsive grid system, 100+ prebuilt components, 500+ incons, and powerful JavaScript plugins.

Get started

100+ Components

More than 100 reusable components for all kinds of frontend tasks: navigation, alerts, dropdowns, inputs and much more

Dependency free

MetroUI library has no dependencies on any third-party libraries. Everything you might need is included in the toolkit.

Technical support (paid option)

We will be there for you when you need help. Rapid response and quick fixing of detected problems.
