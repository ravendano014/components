About Animation

Metro UI contains classes and routines to create different animation effects. All classes stored in metro-animation.css or built in metro-all.css. To use animation classes, you must include file metro-animation.css in head of your page or use metro-all.css. 


                <!DOCTYPE html>
                <head>
                    ...
                    <link href="path_to_metro/metro.css" rel="stylesheet">
                    <link href="path_to_metro/metro-animation.css" rel="stylesheet">
                    ...
                </head>
or

                <!DOCTYPE html>
                <head>
                    ...
                    <link href="path_to_metro/metro-all.css" rel="stylesheet">
                    ...
                </head>
