Flip Card

Use builtin animation for your object to create flip card animation.

About

To create flip-card animation you must create container with class flip-card and add two inner blocks with classes .front and .back. Also you must define size for flip-card: height and width 


                    <div class="flip-card" style="height: 340px; width: 480px">
                        <div class="front"></div>
                        <div class="back"></div>
                    </div>

Activation effect

To activation flip-card effect, you must add one of two subclasses:

  • .effect-on-hover - card flipped on user hover cursor on the card
  • .effect-on-active - card flipped on toggle class .active on card

effect-on-hover


                    <div class="flip-card effect-on-hover">
                        <div class="front"></div>
                        <div class="back"></div>
                    </div>

effect-on-active


                    <div class="flip-card effect-on-active">
                        <div class="front"></div>
                        <div class="back"></div>
                    </div>

Horizontal flip

By default, card flipped in vertical plane. To flip card in horizontal plane, add subclass .flip-horizontal to card. 


                    <div class="flip-card effect-on-hover flip-horizontal">
                        <div class="front"></div>
                        <div class="back"></div>
                    </div>
