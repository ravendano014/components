ViewportCheck

Check if your element in viewport or no with simple component viewport-check.

About

New in 4.4.1

If you add role viewport-check (functional $(el).viewportcheck()) to element, you can checks: your element in viewport or no. 


                    <div data-role="viewport-check"></div>

Events

When you scrolls window, element with role viewport-check checks self position in viewport and generate events:

  • onViewport(state) - state is true, if element in viewport
  • onViewportEnter() - event firing when element enters to viewport
  • onViewportLeave() - event firing when element leaves viewport

                    <div data-role="viewport-check"
                            data-on-view-port="onViewport"
                            data-on-view-port-enter="onViewportEnter"
                            data-on-view-port-leave="onViewportLeave"
                    >
                        ...
                    </div>

                    <script>
                        function onViewport(state){
                            if (state) {
                                console.log("Element in viewport");
                            } else {
                                console.log("Element not in viewport");
                            }
                        }

                        function onViewportEnter(){
                           console.log("Element enters to viewport");
                        }

                        function onViewportLeave(){
                           console.log("Element leaves viewport");
                        }
                    </script>
